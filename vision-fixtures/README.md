# 화면 인식 스파이크용 스크린샷 (fixtures)

이 폴더에 아래 이름 그대로 스크린샷을 넣어주세요. 이걸로 `anchor + layout + recognize`를
실제 화면에서 검증합니다. (라이브 캡처/UI는 그 다음 단계)

## 캡처 규칙 (중요)
- **PNG로 저장** (JPG 금지 — 압축 잡티가 템플릿 매칭을 망칩니다).
- **실제 플레이하는 해상도 그대로**, 자르거나 크기 조절하지 말 것.
- **전체 화면 캡처**(또는 최소한 로스트아크 창 전체가 다 보이게). 창 밖 배경이 같이 들어와도 OK
  — 오히려 배경이 바뀌어도 보드를 찾는지(앵커링) 테스트하는 데 도움이 됩니다.

## 넣어줄 파일 (이름 정확히)

| 파일명 | 무엇을 담아야 하나 |
|---|---|
| `01-empty-board.png` | 게임 시작 직후 **빈 보드** (주사위 거의 없음). 빈 칸 모양 기준. |
| `02-midgame-shields.png` | **중반 보드.** 실드 **초록 테두리·빨강 테두리**가 둘 다 보이게. (지난번 보내준 것 같은 컷) |
| `03-my-turn-rolled-die.png` | **내 턴** — 좌측 홀딩박스에 **굴린 주사위**가 놓여 있는 장면. (가능하면 씨앗=1 말고 일반 숫자 주사위도 한 번) |
| `04-bonus-placement.png` | **알까기 보너스 상태** — 상대 라인에 **흰 테두리**(배치 가능)가 들어온 장면. 없으면 비워도 됨. |
| `05-dice-1-seed.png` | **씨앗(=1)** 주사위가 잘 보이는 컷 (홀딩박스든 보드든). |
| `06-dice-faces.png` | 가능하면 **2~6 숫자**와 **실드 양쪽 색**이 골고루 보이는 컷 1장. (02와 겹쳐도 OK) |

추가로 다른 상태가 보이면 `07-...`, `08-...` 처럼 자유롭게 더 넣어줘도 됩니다.
설명이 필요하면 같은 폴더에 `notes.txt`로 메모 남겨주세요.
